Telerik Universal on Windows IoT

View project on GitHub

Summary

This project demonstrates how easy it is to add an amazing UI to your Windows 10 IoT project using Telerik's UI for Windows Universal. Find the accompanying blog post tutorial here.

https://vine.co/v/eXjzEeD53iu

Prerequisites:

Windows 10 IoT

You can run this on any of the Windows IoT devices. However, since it is a UI based project, I chose Raspberry PI 2 which has an HDMI out.

Go here to select your device and download Windows 10 IoT. You will also find comprehensive instructions based on your device.

Windows 10 SDK

You will need v10240 (or greater) of the Windows 10 SDK. If you did not select it when you installed Visual Studio 2015, you can use the standalone SDK installer found here.

Telerik UI for Windows Universal

You'll need to have UI for Windows installed. It can be a trial or dev license. Download UI for Windows Universal here and find the full documentation here.
